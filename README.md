## App example, a lot of forms using Vee-validator and Bootue components

> Bootue + Boostrap + Vee Validator

### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
